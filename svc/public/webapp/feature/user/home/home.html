PAMM Seed V1.3 Start Page - Introduction

Click on the sample tab to see a sample list implementation. On the "Manage Project" page - you will a list of sample data. This list of data comes from the repository and the (mock) DAL service. See service/repository and service/dal.

Technology Stack

  • AngularJS 1.4
  • Angular UI Router (replaces default router)
  • Bootstrap 3
  • Bootstrap Dialog
  • lodash - Javascript utilities library
  • Font Awesome
  • Angular Google Chart

Test Stack

  • Jasmine 2.0 (unit)
  • karma (auto)
  • Protractor (e2e)
  • CucumberJS (e2e)
Note that Karma and protractor require node.js
